Neovim is my favourite text/code/everything editor. I've been using it for a few years now, and I've never looked back. I'm lazy to write a full review, so here's a gallery and a feature list.

Feature List

Most of nvim's power comes from plugins, but I also included some default features so IDE people would know what to expect if they switch. This is by no means an exhaustive list, there are just way too many plugins and features to list them all.

  • Text Editing
    • (neo)vi(m) keybindings (d'uh)
    • Find/replace in file and outside
    • Fuzzy search over a phrase in files (telescope.nvim)
    • Undo tree (undotree)
    • Snappy leaps within code (leap.nvim)
    • Expansion of selection  (builtin)
    • Tree Sitter to collapse and expand blocks (nvim-treesitter)
  • Software Developing
    • Developing
      • Syntactical and semantic highlighting  (builtin) (nvim-lspconfig)
      • Errors, warnings linting  (builtin) (nvim-lspconfig)
      • GoTo definition/declaration/references  (builtin) (nvim-lspconfig)
      • Code Actions  (builtin) (nvim-lspconfig)
      • Language-specific features (running tests, etc.) (rustaceanvim) (Ionide-vim) ...
      • Debugger (nvim-dap) (nvim-dap-ui)
      • Diff (for merging and just diff-ing)  (builtin) (linediff.vim)
      • Sticky scroll (nvim-treesitter-context)
      • Inlay Hints  (builtin)
    • Interactive Scripting
      • Arbitrary REPL (iron.nvim)
      • Arbitrary Jupyter Kernel (jupyter-kernel.nvim)
      • Evaluation in place with Jupyter (magma-nvim)
    • Tooling
      • Terminal Integration
      • LazyGit Integration  (builtin)
      • Git Integration (Blame, etc.) (gitsigns.nvim) (vim-fugitive)
  • Academia & Notes
    • Orgmode: TODOs with weekly/monthy/yearly agenda (orgmode)
    • Orgmode: Rich formatting (orgmode)
    • Markdown preview (notes & charts) (markdown-preview.nvim)
    • Table mode to create markdown and orgmode tables (vim-table-mode)
    • LaTeX: full LSP and preview (vimtex)
  • Miscellaneous
    • Github CoPilot autocompletion in any file (copilot.vim)
    • Keyboard-only snappy window management
    • Fuzzy search over files to go to
    • gf and gx to go over paths
    • Util to pick colors in place (colortils.nvim)
